# dumbhooks
dumbhooks is a collection of scripts used to do funny things to webhooks that u have the url of <br />
rn we feature 2 scripts: `lyrics.py` and `reddit.py` <br />
## lyrics.py
for this script u can simply input the raw lyrics to a song and the webhook will sing the entire song. 
## reddit.py
for this script, the webhook will scrape from many stupid ass subreddits and post a bunch of unhinged stories
